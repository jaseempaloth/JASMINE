Installation
============

Requirements
------------

* Python 3.9+
* JAX and jaxlib

Install from source
-------------------

.. code-block:: bash

   git clone https://github.com/jaseempaloth/JASMINE.git
   cd JASMINE
   uv sync

Optional extras
---------------

.. code-block:: bash

   uv sync --group dev
   uv sync --group examples
   uv sync --all-groups
