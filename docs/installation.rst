Installation
============

Requirements
------------

* Python 3.8+
* JAX and jaxlib

Install from source
-------------------

.. code-block:: bash

   git clone https://github.com/jaseempaloth/JASMINE.git
   cd JASMINE
   pip install -r requirements.txt
   pip install -e .

Optional extras
---------------

.. code-block:: bash

   pip install .[dev]
   pip install .[examples]
   pip install .[all]
